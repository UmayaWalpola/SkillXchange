
# SkillXchange

**SkillXchange** is a web-based platform designed to connect individuals, organizations, and communities through skill sharing, projects, quizzes, and collaborative opportunities. Inspired by real-world skill-bartering practices, SkillXchange aims to make skill development and knowledge exchange accessible, structured, and engaging.

---

## **Features**

1. **Skill Matching**
   - Users can showcase their skills and get matched with others based on complementary or desired skills.
   - Matching algorithm considers user preferences and skill proficiency levels.

2. **Projects**
   - Users and organizations can create and join projects.
   - Facilitates collaboration and practical experience in real-world scenarios.

3. **Quizzes**
   - Interactive quizzes allow users to test and improve their skills.
   - Progress tracking and performance reports are available for personal growth.

4. **Communities**
   - Users can join or create communities around specific skill areas.
   - Encourages discussion, mentorship, and networking.

5. **Feedback & Reporting**
   - Users receive feedback on project contributions and quiz performance.
   - Admins can generate reports for insights and analytics.

6. **User Roles**
   - Roles include: Individual, Organization, Admin, Quiz Manager, Manager, Community Admin.
   - Each role has specific permissions for managing content and interacting with the system.

---

## **Technology Stack**

- **Backend:** PHP (Vanilla, no frameworks)
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Server Environment:** XAMPP (Apache + MySQL)
- **Version Control:** Git + GitHub
